<--Ejecución programa-->

Para comodidad del usuario se ha creado un ejecutable dentro de la carpeta /dis/ de nombre "trabajo", solo es ejecutarlo para poder usarlo en el computador sin necesidad de usar un entorno de desarrollo.


<--Menú del Porgrama-->

Al iniciar el programa, encontrará el siguiente mensaje:


	<---------------------------------------------->


	1.Lista de adyacencia
	2.Matriz de adyacencia
	3.camino más corto dentre 'a' y 'b'
	4.terminar programa
	R/=

Usted dará un número entre 1 y 4 para elegir la opción que usted apetezca, si usted dará algún otro valor el programa dará el siguiente mensaje:

	la respuesta a no se encuentra dentro del rango 1-4

Y después le dará acceso al menú inicial.

Dentro de cada opción al finalizar la ejecución, le pedirá que use la tecla ENTER para volver a menú, el programa no continuará si no usa la tecla enter, si le da una telca antes no hay problema, el programa no explotará.
